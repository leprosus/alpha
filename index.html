Layout

A layout website is been creating by porting and optimizing the code from 960 Grid System. 12 columns, width of the main block is 60px, distance of between columns and rows is 20px.

Base

Base of layout is node with the class class="grid". inside of the node you should locate a container with the class class="row", witch divides the layout by rows.

Containers with the classes class="cell-x" allow you create different combination of columns and rows. 

<div class="grid">
	<div class="row">
		<div class="cell-5">
			...
		</div><!-- cell-5 -->
		<div class="cell-7">
			...
		</div><!-- cell-7 -->
	</div><!-- row -->
</div><!-- grid -->

Nested columns

To create more complex layouts you can embed containers <div class="row">...</div>: 

<div class="grid">
	<div class="row">				<!-- 1 level -->
		<div class="cell-6">
			<div class="row">			<!-- 2 level -->
				<div class="cell-1">...</div>
				<div class="cell-4">...</div>
				<div class="cell-1">...</div>
			</div><!-- row -->
			<div class="row">			<!-- 2 level -->
				<div class="cell-2">...</div>
				<div class="cell-2">...</div>
				<div class="cell-2">...</div>
			</div><!-- row -->
		</div><!-- cell-6 -->
		<div class="cell-6">
			<div class="row">			<!-- 2 level -->
				<div class="cell-6">...</div>
			</div><!-- row -->
			<div class="row">			<!-- 2 level -->
				<div class="cell-3">...</div>
				<div class="cell-3">...</div>
			</div><!-- row -->
		</div><!-- cell-6 -->
	</div><!-- row -->
</div><!-- grid -->

Navigation

Horizontal navigation

To implement the simplest horizontal navigation you need to use list with the class <ul class="nav">...</ul>:

<ul class="nav">
	<li><a href="#">First option</a></li>
	<li class="active"><a href="#">Second is active</a></li>
	<li><a href="#">Third option</a></li>
	<li><a href="#">Last option</a></li>
</ul><!-- nav -->

Active navigation option is selected by the class .active.

As a result we get the following navigation menu:

  • First option
  • Second is active
  • Third option
  • Last option

Horizontal tabs

There is additional class .nav-tabs, witch changes the horizontal navigation menu design:

<ul class="nav nav-tabs">
	...
</ul><!-- nav-tabs -->
  • First option
  • Second is active
  • Third option
  • Last option

Vertical navigation

To create vertical navigation you need to use list with the class .nav-list:

<ul class="nav-list">
	<li><a href="#">First option</a></li>
	<li class="active"><a href="#">Second is active</a></li>
	<li><a href="#">Third option</a></li>
	<li><a href="#">Last option</a></li>
</ul><!-- nav-list -->
  • First option
  • Second is active
  • Third option
  • Last option

Vertical tabs

To create vertical tabs you need to append additional class .nav-tabs: 

<ul class="nav-list nav-tabs">
	...
</ul><!-- nav-tabs -->
  • First option
  • Second is active
  • Third option
  • Last option

Navigation panel

The panel contains two base elements, .nav-bar and nested .body. Inside of the panel you can locate a logo (.title) and navigation (.nav):

<div class="nav-bar">
	<div class="body">
		<div class="title">
			<a href="#">alpha</a>
		</div><!-- title -->
		<ul class="nav">
			<li><a href="#">...</a></li>
		</ul><!-- nav -->
	</div><!-- body -->
</div><!-- nav-bar -->

Bread crumbs

Bread crumbs can be created by using general list with the class <ul class="breadcrumbs">...</ul>:

<ul class="breadcrumbs">
	<li><a href="#">Main</a></li>
	<li><a href="#">Category</a></li>
	<li><a href="#">Nested category</a></li>
	<li class="cur">Product</li>
</ul><!-- f-breadcrumb -->

Page-by-page navigation

The class <div class="pager">...</div> is usage as base for page-by-page navigation. Inside of the container are located links.

«Next» and «Previous» links are separated by the classes .pager.prev and .pager.next. The active element is the class .active.

<div class="pager">
	<ul>
		<li class="pager prev"><a href="#">Prev</a></li>
		<li><a href="#">1</a></li>
		<li class="active"><strong>2</strong></li>
		<li><a href="#">3</a></li>
		<li class="pager next"><a href="#">Next</a></li>
	</ul>
</div><!-- pager -->
