Макет страницы

Макет (layout) сайта создается с помощью портированного и оптимизированного кода из 960 Grid System. 12 колонок, основной блок шириной 60px, расстояние между колонками и рядами 20px.

Основа

Основой разметки служит родительский блок с классом class="grid". В нём следует расположить контейнер с классом class="row", который разделяет разметку на ряды.

Контейнеры с классами class="cell-x" позволяют создавать различные конфигурации колонок в рядах. 

<div class="grid">
	<div class="row">
		<div class="cell-5">
			...
		</div><!-- cell-5 -->
		<div class="cell-7">
			...
		</div><!-- cell-7 -->
	</div><!-- row -->
</div><!-- grid -->

Вложенные колонки

Для создания более сложных разметок вы можете вкладывать контейнеры <div class="row">...</div> друг в друга:

<div class="grid">
	<div class="row">				<!-- 1 уровень -->
		<div class="cell-6">
			<div class="row">			<!-- 2 уровень -->
				<div class="cell-1">...</div>
				<div class="cell-4">...</div>
				<div class="cell-1">...</div>
			</div><!-- row -->
			<div class="row">			<!-- 2 уровень -->
				<div class="cell-2">...</div>
				<div class="cell-2">...</div>
				<div class="cell-2">...</div>
			</div><!-- row -->
		</div><!-- cell-6 -->
		<div class="cell-6">
			<div class="row">			<!-- 2 уровень -->
				<div class="cell-6">...</div>
			</div><!-- row -->
			<div class="row">			<!-- 2 уровень -->
				<div class="cell-3">...</div>
				<div class="cell-3">...</div>
			</div><!-- row -->
		</div><!-- cell-6 -->
	</div><!-- row -->
</div><!-- grid -->
